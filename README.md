# TaskManager

## Project Description
This project is a simple task management application where users can add, view, complete, and delete tasks.

## Final Goals
By the final submission, I plan to have a working task manager that allows users to create tasks, mark them as completed, delete tasks, and save their task list.
# Sprint 3

## Final Demonstration
For the final submission, I plan to demonstrate a console-based Task Manager application that allows users to:
- Add tasks
- View tasks
- Complete tasks
- Delete tasks
- Manage tasks through a simple menu-driven interface

## Problems Encountered
Some challenges included organizing the project structure, testing the application, and integrating the required design patterns into the project.
